* Introducción

** Resumen

+ Apuntes de lectura de [[https://leanprover-community.github.io/mathematics_in_lean/index.html][Mathematics in Lean]] de Jeremy Avigad, Kevin Buzzard,
  Robert Y. Lewis y Patrick Massot.

+ Su objetivo es presentar el uso de Lean mediante ejemplos matemáticos usando
  tácticas y mathlib.
